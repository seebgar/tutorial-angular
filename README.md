# Introduction

## Goal

* `ECMAScript` &rArr; `TypeScript` &rArr; `Angular` 순서로 학습합니다.
* 기초를 소홀히 하지 않습니다. 
* 암기는 지양하고 흐름을 지향합니다.
* 개념은 되도록 간단하게 정리하고 코드를 통해서 원리를 이해합니다.
* 실무에 적용할 수 있는 코드를 통해서 배우고 익혀 바로 프로젝트에 적용합니다.
* 앵귤러와 사랑에 빠지게 되고 다른 개발자들에게 추천하는 행동을 보입니다.

## Version

* `OS`: win32 x64
* `Node`: 10.14.1
* `Angular CLI`: 7.1.1
* `Angular`: ~7.1.0
* `typescript`: 3.1.6
